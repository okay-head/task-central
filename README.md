## Install

This project is built with

```
pnpm v8.12.0
node v20.7.0
```

1. If you want to install this repo locally.
   Fork and pull this repo, then run

```sh
pnpm install
```

&nbsp;&nbsp;&nbsp;&nbsp;or

```sh
npm install
```

2. Spin up the development server

```sh
pnpm dev
```
